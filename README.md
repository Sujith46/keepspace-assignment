## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Tech stack and libraries

- NextJS
- Tailwind CSS
- Daisy UI
- Axios
- React redux
- Redux toolkit
- React infinite scroll component
